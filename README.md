## 🚀 Projects Included

### 1. 🍰 Desserts Product List with Cart Functionality
A fully responsive, interactive dessert ordering page built from a Figma design challenge.
* **Key Features:**
  * Dynamic grid layout for products that adapts from desktop (3 columns) to mobile (1-column stack).
  * Fully functional Cart system (Add to cart, increase/decrease quantity, and remove items via custom Figma-style SVG cross button).
  * Real-time price and item count synchronization.
  * Order confirmation modal popup with order summary and reset options.
* **Technologies Used:** HTML5, CSS3 (Media Queries, Grid, Flexbox), Vanilla JavaScript (Async/Await, Fetch API).
